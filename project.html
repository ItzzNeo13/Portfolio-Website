PROJECTS -

Minimal Theme:

This is a visually stunning theme designed for Visual Studio Code, featuring a warm color palette that creates a cozy and welcoming ambiance. The theme is optimized for dark mode, providing a smooth and soothing experience for long hours of coding.

Url Reaper Discord Bot:

Url Reaper is a bot that was developed to help protect users from being hacked by spam nitro links on Discord servers. The bot has been archived and is no longer maintained, but its legacy lives on.

Neoquiz:

The Command Line Quiz Game is a fun and interactive quiz game that runs on the command line. The game is built using Node.js, a popular and powerful JavaScript runtime environment that lets developers create applications.

React E-Commerce Site

The "React Ecommerce Site" Github repository is a web development project that aims to create an e-commerce website using React, a popular front-end JavaScript library.

React Bored Tasks

The "Bored Tasks" project is a web application built using the React.js framework and powered by the BoredAPI. This application aims to help people find fun and engaging activities to do when they are bored.

React Todo List

The "React Todo List" Github repository is a web development project that aims to create a simple yet effective to-do list application using React. Does not yet use Local Storage but will roll out this feature soon!
